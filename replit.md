# Website Profile - I Putu Indrawan

## Overview
Website portfolio sederhana yang menampilkan profil I Putu Indrawan (Prompt Engineer). Website ini dibuat berdasarkan desain dari https://siputzx.my.id/.

## Struktur Project
- `index.html` - Halaman utama website dengan desain lengkap
- `server.js` - Simple HTTP server untuk menjalankan website
- `package.json` - Konfigurasi Node.js project
- `favicon.png` - Icon tab browser dengan tema coding purple

## Fitur Website
- Profil lengkap dengan foto
- Link ke online identities (GitHub profiles)
- Informasi personal
- Daftar skills
- Portfolio projects dengan links
- Favicon custom dengan tema purple coding

## Teknologi
- HTML5
- CSS3 (dengan animasi dan responsive design)
- Node.js HTTP server
- Background gradient purple modern

## Cara Menjalankan
Website sudah berjalan otomatis dengan workflow yang sudah dikonfigurasi. Server berjalan di port 5000.

## Desain
- Background: Dynamic time-based images (morning, afternoon, evening, night)
- Layout: Glassmorphism cards with blur effect
- Responsive design untuk mobile dan desktop
- Animasi hover effects
- Color scheme: Purple (#667eea, #764ba2)
- Frosted glass effect dengan backdrop-filter blur
